## Description
A3M Router is the **#1 ranked LLM router** on RouterArena (76.43 score) and the **cheapest** at $0.047/1K queries.

While other routers try providers one-by-one (sequential fallback), A3M runs multiple providers simultaneously with confidence-scored voting. This gives you:

- 🏆 **Best accuracy** — #1 on RouterArena, beating Microsoft Azure and OpenAI GPT-5
- 💰 **Lowest cost** — 213× cheaper than GPT-5, 4× cheaper than #2
- ⚡ **Low latency** — parallel execution, no sequential wait
- 🧠 **Memory** — the only LLM router with episodic memory
- 🔄 **47+ providers** — OpenAI, Anthropic, Groq, DeepSeek, NVIDIA, Together, and more

Install in 5 seconds:
```
npx a3m-router route "your query"
```
